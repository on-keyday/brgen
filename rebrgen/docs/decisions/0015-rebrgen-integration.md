# rebrgen を brgen リポジトリに統合した

## 日付

- 判断時期: 2026年3月11日（subtree として brgen リポジトリに統合）
- 文書化: 2026-04-15

## 判断

rebrgen を別リポジトリから brgen の `rebrgen/` サブディレクトリに統合した。

## 動機

- EBM の仕組みが成熟してきた結果、EBM の変更のために brgen と rebrgen の
  両方にまたがって変更が必要なケースが出てきた。
- それまでは brgen の開発を一時的に停滞させて rebrgen 側の作業を進めていたが、
  変更の逆流が起こるようになったため統合のタイミングと判断した。

## これは X を意味しない

- brgen と rebrgen が一体化したわけではない。ディレクトリ構造やビルドは依然として分離している。
